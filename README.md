# Echoes of Eternity

A time-loop survival game prototype built in Godot 4.

## Description
You are trapped in an eternal loop. Explore an infinite procedural world, gather relics to enhance your abilities, and build structures that persist across time loops. Uncover the secrets of the loop and achieve eternity.

## Features
-   **Time Loop Mechanic**: Reset the world but keep your stats and inventory.
-   **Persistence**: Structures you build and relics you collect remain across loops.
-   **Procedural World**: Infinite terrain generation with biomes.
-   **Relic System**: Collect relics to increase speed and jump height.
-   **Enemies**: Avoid or outrun the chasers that hunt you down.
-   **Cozy Hub**: A safe zone at (0, 50, 0) to rest and reflect.
-   **Ghost Trail**: See the path of your past self.

## Controls
-   **WASD**: Move
-   **Space**: Jump
-   **Shift**: Sprint
-   **Mouse**: Look
-   **B**: Build Mode (Place Persistent Object)
-   **T**: Trigger Time Loop (Reset)
-   **Esc**: Pause / Menu

## Win Condition
-   Reach **Loop 12**
-   Collect **25 Relics**
